# Assets Directory
Coloque aqui as imagens exportadas para uso via importação estática no Vite.
Para garantir o enquadramento perfeito, utilize as classes Tailwind `object-cover` e `object-center` (ou `object-top` para certificados).
As imagens atuais estão sendo servidas pela pasta `public/evidencias` para manter a estabilidade enquanto os arquivos binários não são anexados ao sistema de arquivos.
